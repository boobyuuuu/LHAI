CodeNote for LHAI

2. 启动服务器 VPN 加速：

source /etc/network_turbo

3. 下载 LHAI 文件：

git clone --branch Code https://github.com/boobyuuuu/LHAI.git

4. 安装环境 & 上传数据：

cd LHAI
pip install -r requirements.txt

5. 处理数据：

python main_dataprocess_gpu.py \
  --input_path /root/LHAI/data/Train/xingwei_10000_64_train_v1.npy \
  --output_path /root/LHAI/data/Train/xingwei_10000_64_train_v1_processed.npy \
  --num_to_learn 10000 \
  --data_range 1.0 \
  --remove_duplicates \
  --augment \
  --enhance \
  --remove_low_info \
  --device cuda

6. 使用默认数据 xingwei 训练：

python train_cnn.py

7. 使用处理后数据训练：

python train_cnn.py \
  --data-name "xingwei_10000_64_train_v1_processed.npy" \
  --exp-name "dataprocessed"

8. 使用模型1进行评估：

python evaluation.py

9. 使用模型2进行评估（示例）：

python evaluation.py --model-name model_v2.pth

10. 上传至 GitHub：

git add .
git commit -m "..."
git push
